# davetnerede.com Platformu Yapım Todo Listesi

Davet burada isimli bir platform yapacağız. Bu platformun amacı kullanıcı adres araması yapacak ve ara diyecek, dave salonları listelenecek. Kullanıcı istediği davet salonların profiline girebilir. Profilde davet salonu hakkında tüm bilgi yer almalı. 

Davet salonundan teklif alma butonu olmalı. Teklif al butonuna tıklayınca kullanıcıya bir modal açmalıyız ve kullanıcı hakkında temel ve gerekli bilgileri toplamalıyız. 

Liste sayfasında filtre butonu olmalı. Firma listesinden Teklif Al ile teklif isteyebilir. Salonda detayları kısmında işletmenin detaylı iletişim ve adres bilgisi yer almalı. 

Proje teknik detayları:

### **Temel Teknoloji Yığını**

1. **Next.js**: Modern React framework
- Server-side rendering, statik site oluşturma
- App Router ve sayfa yönlendirme
- API Routes ile backend entegrasyonu
1. **React**: UI bileşenlerini oluşturmak için
- Functional bileşenler
- React Hooks (useState, useEffect, useContext)
- Client-side rendering
1. **TailwindCSS**: Hızlı ve kolay stil oluşturma
- Utility-first yaklaşım
- Responsive tasarım
- Kolay özelleştirme

### **UI Bileşen Kütüphaneleri**

1. **Shadcn UI**: Hazır, özelleştirilebilir bileşenler
- Doğrudan projeye kopyalanabilen bileşenler
- Radix UI temelli ancak styled components
- Dialog, Button, Card, Input gibi temel bileşenler
1. **Radix UI**: Primitives
- Dialog, Popover, Select, Checkbox vb.
- Erişilebilirlik odaklı
- Styled-agnostic
1. **Lucide React**: Modern, minimal ikon kütüphanesi
- 1000+ ikon
- Kolay entegrasyon
- SVG temelli

### **Veri ve State Yönetimi**

1. **Supabase**: Firebase alternatifi
- PostgreSQL veritabanı
- Authentication
- Storage çözümü
- Real-time subscriptions
1. **Redux Toolkit**: Global state yönetimi
- createSlice API
- Redux Thunk entegrasyonu
- Basitleştirilmiş Redux deneyimi

### **Styling Yardımcıları**

1. **clsx/tailwind-merge**: Class'ları birleştirme
- Koşullu sınıf isimleri
- Tailwind sınıflarını birleştirme
1. **class-variance-authority**: Bileşen varyantları
- Farklı bileşen durumları için stilleme
- Tailwind ile uyumlu

Supabase ve veritabanı bağlantılarını daha sonra yapacağız. İlk önce statik olarak sabit datalar ile tasarımı iyileştirelim.
